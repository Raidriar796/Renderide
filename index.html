Open-source renderer

Renderide

Renderide is an experimental Rust and wgpu renderer for Resonite. It runs alongside the existing host process and focuses on cross-platform rendering, OpenXR, shared-memory IPC, and renderer architecture that can evolve independently.

View the GitHub repository AUR package

About

The project is a drop-in renderer replacement written in Rust on top of wgpu. Renderide targets Linux, macOS, and Windows, with renderer systems for assets, materials, cameras, render graph execution, profiling, and VR presentation.

Highlights

Rust and wgpu

Built around portable graphics backends including Vulkan, Metal, and DirectX 12 through wgpu.

OpenXR-first VR

VR presentation is part of the renderer path instead of being a separate afterthought.

Renderer architecture

Frontend, scene, backend, and runtime layers separate host IPC from GPU work and frame orchestration.

Repository

Source code, build instructions, issue tracking, releases, and contribution details live at github.com/DoubleStyx/Renderide. Clone it with git clone https://github.com/DoubleStyx/Renderide.git.

Renderide is an unofficial renderer project for Resonite. Resonite is a separate project owned by Yellow Dog Man Studios.
